# 分镜截图清单 / Screenshots needed

把下面 9 张 PNG 放进这个文件夹（文件名严格对应），打开 `/storyboard.html` 即可看到效果。

## 截图方法（推荐）

Chrome → DevTools (⌥⌘I) → 点切换设备按钮 → 选 **iPhone 12 Pro (390×844)** 或自定义 **414×896** → 在 dev server 下走到对应场景 → DevTools 右上 ⋮ → **Capture screenshot**（截当前可视区域，不要 full size）。

或在浏览器 zoom 到合适大小手动截图后裁剪成 9:16 比例即可。

每张建议长边 ≥ 1200px，PNG 优先（保留锐边）。
